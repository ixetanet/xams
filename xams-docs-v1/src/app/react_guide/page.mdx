# React

React components are built using [Mantine v7](https://mantine.dev/) {{ className: 'lead' }}

React component examples and documentation can be found [here](https://examples.xams.io/) {{ className: 'lead' }}

## React Data types

The React components are compatible with the following field types. Interaction with fields of other data types may not function properly when using the included React components.

- String
- Float
- Double
- Int32
- Int64
- Decimal
- Guid
- DateTime
